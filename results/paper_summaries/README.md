# Paper Summary Artifacts

This directory contains compact summaries of the final COPAL experiments used for paper tables.

Included summaries:

- `table2_ablation/table2_summary.json`
- `table2_ablation/hidden_info_repair_manifest.json`
- `table3_model_eval/table3_summary.json`
- `paired_single_composed/paired_single_composed_summary.json`
- `judge_sensitivity/judge_sensitivity_summary.json`
- `judge_sensitivity/judge_disagreement_analysis.json`

These are summary artifacts, not full raw-response logs. The raw long-running experiment directories are excluded from the public release because they are large and include provider-cache/runtime details.
